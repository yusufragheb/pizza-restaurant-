# 🍕 Pizza Style – Food Ordering Website

A simple **pizza ordering website** built using **HTML, CSS, and JavaScript**.
The project allows users to log in and choose their favorite pizza from a menu and place an order.

This project is a basic front-end practice for learning web development concepts.

---

# 📌 Features

* Simple **login page**
* Pizza menu with different options
* Display pizza ingredients
* Add pizza to cart
* Show order details
* Show total price
* Reset order option

---
